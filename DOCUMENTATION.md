# Documentation

## Project Structure

- main.py → Main program
- README.md → Project overview
- CONTRIBUTING.md → Contribution guide
- DOCUMENTATION.md → Detailed explanation

## Features

### Task Creation
Users can add tasks with:
- Title
- Subject
- Deadline
- Priority

### Task Management
- View all tasks
- Mark tasks as completed

### Report Generation
The system generates:
- tasks_report.txt → Text summary
- tasks.csv → CSV format data

## How It Works

1. Run the program
2. Choose an option from the menu
3. Add or manage tasks
4. Generate reports

## Output Files

### tasks_report.txt
Contains a readable summary of all tasks

### tasks.csv
Stores tasks in structured format for further analysis
